### Featured Projects

#### 1. HTML-to-Others Conversion Service
A lightweight Next.js service that handles high-fidelity document conversion.
* **The Core:** Converts user-provided HTML into downloadable PDF and DOCX files.
* **How it works:** Uses Puppeteer for server-side PDF rendering. Features a dual-flow DOCX system: a lightweight `html-to-docx` parser for speed, and a LibreOffice pipeline for complex, high-fidelity documents.
* **Stack:** Next.js, Node.js, Puppeteer, LibreOffice API.

#### 2. React Expense Tracker
A focused, client-side application built to master complex UI state management.
* **The Core:** Tracks personal finances with real-time budget updates.
* **How it works:** Built purely using React state and hooks (no external heavy libraries) to maintain predictable data flow and crisp UI performance.
* **Stack:** React, Custom CSS.
